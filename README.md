#  Law Guard System

**Проект Law Guard** — это прототип ERP-системы для автоматизации юридических услуг. Цель: обеспечить прозрачную связь между клиентами, юристами и заказами с расчетом продуктовых метрик.

##  Карта проекта (Навигация)
*    **[Бизнес-анализ и User Stories](docs/BUSINESS_ANALYSIS.md)** — описание ролей и потребностей пользователей.
*    **[Техническая архитектура (SQL)](sql-scripts/init_db.sql)** — схема базы данных со всеми связями.
*    **[Аналитические отчеты](sql-scripts/analytical_views.sql)** — запросы для расчета выручки и среднего чека (AOV).
*    **[Тестовые данные](sql-scripts/seed_data.sql)** — скрипт для наполнения базы.

##  Схема базы данных (PostgreSQL)
- **Таблица Clients**: реестр контрагентов.
- **Таблица Lawyers**: база квалифицированных защитников.
- **Таблица Orders**: транзакционный слой с Foreign Keys.
