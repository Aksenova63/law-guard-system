# Бизнес-анализ проекта Law Guard

## User Stories (Пользовательские истории)

1. **Как Клиент (владелец бизнеса)**, я хочу иметь возможность выбрать юриста по его рейтингу и стажу, чтобы быть уверенным в качестве защиты.
2. **Как Юрист**, я хочу видеть список своих активных заказов и их стоимость, чтобы эффективно планировать свою нагрузку и доход.
3. **Как Администратор системы**, я хочу видеть общую выручку и средний чек (AOV) по дням, чтобы оценивать темпы роста сервиса.
