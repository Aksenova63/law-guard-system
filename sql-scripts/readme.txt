#SQL Scripts for Law Guard System

###Порядок запуска скриптов для развертывания базы данных:

1. **init_db.sql** — Создание структуры таблиц (Схема).
2. **seed_data.sql** — Наполнение справочников и создание тестовых заказов (Данные).
3. **business_queries.sql** — Аналитические выборки и финансовые отчеты (Логика).
